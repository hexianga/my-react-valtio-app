- 2. 标注:1. 划选文本实现高亮 2. 基于 Canvas 绘制任意形状进行标注，包括不规则多边形，圆，矩形等
- 3. 导出:支持将标注的内容导出至 Flowus 的⻚面进行摘要汇总
- 4. 下载:支持在下载 PDF 时将标注一同下载
